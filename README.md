# Flipside

# Flipside ist eine Karteikarten-Webanwendung zum Erstellen und Abfragen von Lerndecks. Karten lassen sich per Klick umdrehen – Frage auf der Vorderseite, Antwort auf der Rückseite. Decks sind frei nach Thema und Kategorie organisierbar.

#Lernkarten-App
Diese Anwendung ermöglicht es Nutzerinnen und Nutzern, eigene Karteikarten-Decks zu beliebigen Themen zu erstellen und effektiv zu lernen. Jedes Deck besteht aus individuell gestaltbaren Karten mit Fragen und Antworten, die jederzeit bearbeitet oder erweitert werden können.
